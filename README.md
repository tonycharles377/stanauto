# Stan Auto Center — Landing Page

A responsive business landing page built for **Stan Auto Center**, a professional auto garage based in Nairobi, Kenya. This is a front-end project built with pure HTML and CSS as part of a web development learning journey.

---

## 🚗 About the Project

This landing page is designed to give Stan Auto Center a strong online presence — helping the business attract new customers, showcase its services, and make it easy for people to get in touch.

The page is built with a **mobile-first development philosophy**, ensuring a great experience for the majority of Kenyan users who browse on mobile devices.

---

## 📄 Pages & Sections

- **Navigation Bar** — Fix nav with smooth scroll links to all sections
- **Hero Section** — Bold headline, subtext, and a call-to-action button
- **About Us** — Business story and values
- **Our Services** — Six service cards covering the garage's full offering
- **Why Choose Us** — Four value proposition cards with supporting testimonials
- **Contact Form** — Validated form with name, phone, email, service selection, and message fields
- **Footer** — Business info and copyright

---

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Mobile-first styling, Flexbox, Grid, and media queries

> No JavaScript frameworks or libraries were used. This is a pure HTML/CSS project.

## ✅ Features

- Mobile-first responsive design
- Semantic HTML5 elements (`<nav>`, `<section>`, `<footer>`, `<blockquote>`)
- Accessible form with labels, `autocomplete`, `required` attributes, and placeholder text
- Service selection using `<datalist>` for a better UX
- Hover effects on buttons and cards
- Clean, consistent color palette and typography

---

## 🚀 How to Run

No installation or build tools needed.

1. Clone or download the repository
2. Open `index.html` in any modern browser

```bash
git clone https://github.com/your-username/stan-auto-center.git
cd stan-auto-center
open index.html
```

## 🎯 Purpose & Learning Goals

This project was built to cement foundational HTML and CSS skills, including:

- Writing semantic, accessible HTML
- Structuring multi-section landing pages
- Applying mobile-first CSS with Flexbox and Grid
- Using media queries to scale layouts across screen sizes
- Building and validating HTML forms

It also serves as the foundation for future JavaScript features, including form validation, interactive service filtering, and booking functionality.
